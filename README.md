# sec-review

A Claude Code plugin that performs citation-grounded cybersecurity reviews of web services and servers. Pairs a domain-expert subagent with live CVE feeds (NVD, OSV.dev, GitHub GHSA) to produce a prioritized report of reliable, cited fixes.

> This README is a stub. The full install / quick-start / scope / limits sections are written in Stage 5 of the plan.

## Install (preview)

```text
/plugin marketplace add /home/user/dev/sec-review
/plugin install sec-review
/sec-review <path-to-project>
```

See `docs/plans/2026-04-21-sec-review-plugin-plan.md` for the staged build plan.
